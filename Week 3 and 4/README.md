# Week 3 and 4: CodeX Beverage Pricing Project

This phase contains the CodeX project, an end-to-end machine learning workflow for predicting the likely price range a consumer will accept for an energy drink product.

## What is covered

- survey data cleaning
- feature engineering for modeling
- classification model comparison
- MLflow experiment tracking
- serialized model packaging
- FastAPI inference service
- Streamlit demo application

The main project lives in `CodeX Project/`.
