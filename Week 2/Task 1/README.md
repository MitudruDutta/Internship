# Week 2 Task 1: Statistical Analysis of E-commerce Data

This task answers a set of client questions using descriptive statistics, probability, and inferential statistics on an e-commerce customer dataset.

## Objective

The notebook is built to compute exact statistical answers from the provided dataset instead of producing a dashboard or machine learning model.

## Topics covered

`task1.ipynb` includes:

- range of customer age
- mean, median, and mode of amount spent
- variance and standard deviation of time spent on site
- interquartile range of amount spent
- correlation between purchases and time spent
- z-score for a specific spending amount
- skewness of amount spent
- probability for purchase and spending thresholds
- 95 percent confidence interval for amount spent
- conditional probability of high cross-sell conversion

## Files in this directory

- `task1.ipynb`: main notebook
- `datasets/ecommerce_data.csv`: input dataset
- `meta_data.txt`: data definitions
- `client_requests.pdf`: original task prompt

## How to run

1. Install dependencies from the repository root with `pip install -r requirements.txt`.
2. Start Jupyter with `jupyter notebook`.
3. Open `task1.ipynb` and execute the notebook from top to bottom.
